# Art, Sound & Music Skill

## Mission
Guide aesthetic direction so visuals, audio, and music reinforce narrative tone even before full production ramps.

## Workflow
1. Create lightweight moodboards or reference callouts tied to each level or character arc.
2. Define palettes, audio motifs, and accessibility considerations (contrast ratios, audio cues, captions).
3. Document implementation hooks (resource paths, bus routing, shader needs) for engineers.
4. Capture placeholder asset needs and track them as TODO tickets for future sprints.

## Best Practices
- Keep style guides modular so contributors can mix cozy warmth with tactical clarity.
- Use descriptive naming for assets to aid automation scripts.
- Note licensing or attribution requirements early.

## Collaboration
- Align with Narrative skill for emotional beats and with Product/Customer skills for priority cues.
- Signal QA when audiovisual changes may affect performance or UX tests.
- Leave breadcrumbs (comments or TODOs) when dependent on unbuilt systems to avoid blocking others.
